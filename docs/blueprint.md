# **App Name**: Color Clash

## Core Features:

- Countdown Timer: Display a countdown timer that resets every 30 seconds, indicating the start of a new round.
- Color Betting Buttons: Enable users to place bets on Red, Green, or Violet, with the odds clearly displayed for each color. Display current round's bets.
- Betting History Tab: Display a tab to switch between current round bets and the complete betting history.
- Real-time Balance Update: Display real-time balance updates, showing current funds available for betting.
- AI Winning Color Selection: Provide an AI-driven tool mode to determine the winning color for each round based on which color has the least amount bet on it.
- Backend Socket Handling: Backend handling of real-time socket communication and processing of user bets

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to establish trust and tranquility, crucial for a betting interface.
- Background color: Soft gray (#F0F4F8), a muted tone that will provide contrast while remaining visually restful.
- Accent color: A muted purple color (#9575CD), inspired by violet and to signal for focus on call-to-action buttons.
- Font for body and headline: 'Inter' (sans-serif), will create a neutral and machined look.
- Use simple, outlined icons to represent different betting options and game states, enhancing clarity without overwhelming the interface.
- Divide the screen into clear sections: the betting area, the timer display, the balance indicator, and the betting history tab. Maintain a balanced and consistent distribution of elements.
